# AI Tool Hub — AI 工具导航与实用工具合集站

## 产品定位
面向中文用户的 AI 工具导航站 + 实用在线小工具合集。每天自动抓取并更新 AI 工具榜单，沉淀长尾 SEO 流量，通过广告联盟 + AI 工具 CPS 分销变现。

## 目标用户
- 想找 AI 工具但不知道有哪些的普通用户
- 需要"在线 XX 工具"的长尾搜索流量
- 关注 AI 工具动态的从业者

## 核心模块（MVP）
1. **首页** — 工具分类导航 + 热门工具推荐 + 最新动态
2. **AI 工具榜单页** — 按类别（写作/绘图/编程/视频/音频）分类的 AI 工具列表，含简介、价格、链接
3. **工具详情页** — 单个工具的详细介绍 + 使用入口（CPS 跳转）
4. **实用小工具** — 在线工具集合（Markdown 预览、字数统计、UUID 生成等），纯前端实现，零服务器成本

## 技术栈
- **框架**：Astro（静态优先，SEO 友好）
- **样式**：Tailwind CSS
- **部署**：GitHub Pages（零成本）
- **数据**：JSON 文件存储工具数据，由 AI 每日自动更新提交

## 变现路径（按起量顺序）
1. **流量起步期（0-1k UV/天）**：暂不接广告，专注 SEO 内容沉淀
2. **流量起量期（1k+ UV/天）**：接入 Google AdSense（需用户实名注册一次）
3. **稳定期**：AI 工具 CPS 联盟分销（Notion、Figma、各类 AI 产品的推广分佣）
4. **长期**：部分高级工具付费解锁（小额订阅/一次性付费）

## SEO 策略
- 每个工具详情页独立静态 HTML，搜索引擎可索引
- 结构化数据（JSON-LD）标注工具信息
- 自动生成 sitemap.xml
- 长尾关键词覆盖：每个工具名 + "中文/教程/对比/替代品" 等修饰词

## 自动化运行（核心）
- **每日**：AI 抓取 AI 工具新闻/榜单 → 更新 tools.json → 提交到 GitHub → 自动部署
- **每周**：AI 生成 SEO 优化内容（工具对比、教程、榜单解读）
- **每月**：AI 输出数据报告，用户每周看 10 分钟即可

## 一次性用户介入事项（清单）
- [ ] 确认 GitHub 账号（已完成：azri57806-design）
- [ ] 注册 Google AdSense（流量起来后）
- [ ] 注册 AI 工具联盟账号（如 Notion Partner 等）
- [ ] 可选：购买自定义域名（前 3 个月用 GitHub Pages 默认域名即可）
